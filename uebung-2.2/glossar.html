Glossar

A | B | C | D | E | F | G | H | I | J | K | M | O | P | Q | R | S | T | U | V | W | X

A

Aliasing
Treppenförmige Kanten, die zwischen den Farben einer Bitmapgrafik auftauchen können.
Anker
Das zum Erstellen von Hyperlinks verwendete HTML-Element.
Applet
Ein kleines, selbständiges Programm, z.B. in der Programmiersprache Java geschrieben.
Attribut
Parameter, die ein HTML-Element erweitern oder modifizieren.

B

Backend
Aspekte einer Website, die solche Funktionen betreffen, die auf dem Server ablaufen.
Bittiefe
Ein Mass für Farben, das auf der Anzahl der Bits beruht, die von der Datei oder dem System zugewiesen werden
Browser
Die Client-Software, die Webseiten anfordert und anzeigt.

C

Cache
Temporärer Speicher, den Browser für heruntergeladene Dokumente verwenden.
CGI
Common Gateway Interface, ein Mechanismus zur Kommunikation zwischen dem Webserver und anderen Programmen, die auf dem Server laufen
Codec
Ein Algorithmus, der von Mediendateien zum Komprimieren und Dekomprimieren von Daten verwendet wird.

D

Dithering
Die Ersetzung bzw. Annäherung einer Farbe durch eine Mischung ähnlicher Pixel, die in der Bild- oder Systempalette zur Verfügung stehen.
Document Type Definition (DTD)
Ein XML-Dokument, das alle Elemente und Attribute einer Auszeichnungssprache definiert.
Domainname
Ein Name, der einer bestimmten IP-Adresse entspricht.

E

Extranet
Eine Website oder ein System, das innerhalb eines geschlossenen Systems und für ausgewählte Benutzer ausserhalb dieses Systems zur Verfügung steht, meist über ein Passwort.

F

Flash
Ein Multimedia-Format für die Bereitstellung von Animationen, Interaktivität und Audio im Web.
Frame-Rate
In Videos und Animationen die Anzahl der Frames pro Sekunde.
FTP
File Transfer Protocol; ein System zum Verschieben von Dateien über das Internet von einem Computer zum anderen.

G

Gamma
Bezieht sich auf die Gesamthelligkeit der Monitoranzeige eines Computers.
GIF
Graphic Interchange Format; ein verbreitetes Dateiformat für Webgrafiken. GIF ist ein palettenbasiertes 8-Bit-Format.

H

Hexadezimal
Ein Zahlensystem zur Basis 16, das die Zeichen 0-9 und A-F verwendet. Es wird in CSS und HTML zum Angeben von Farbwerten benutzt.
Host
Ein anderer Begriff für den Server. Hosting-Dienstleister sind Firmen, die Platz für Websites bereitstellen.
HTML
HyperText Markup Language; die Auszeichnungssprache, mit der man die Struktur von Webdokumenten festlegt.
HTTP
Hypertext Transfer Protocol; System, das definiert, wie Webseiten und Medien angefordert und zwischen Servern und Browsern übertragen werden.

I

Imagemap
Ein Bild, das mehrere Hypertext-Links enthält.
IP-Adresse
Ein numerischer Identifikator für einen Computer oder ein Gerät in einem Netzwerk.
ISP
Internet Service Provider; ein Unternehmen, das Zugang zum Computernetzwerk Internet verkauft.

J

Java
Eine plattformunabhängige, objektorientierte Programmiersprache, entwickelt von Sun Microsystems.
JavaScript
Eine Skriptsprache, mit der man Elementen auf Webseiten und Browsern bestimmte Verhaltensweisen hinzufügen kann.

K

Komprimierung
Eine Methode zum Reduzieren der Dateigrösse eines Dokuments. Die verschiedenen Dateiformate benutzen unterschiedliche Komprimierungstechniken.

M

MP3
Ein beliebtes Dateiformat für hochwertiges Audio, das die MPEG-Komprimierung verwendet.
MPEG
Eine Familie von Multimedia-Standards, geschaffen von der Motion Picture Experts Group

O

Optimieren
Reduzieren der Dateigrösse, um die übertragungszeit über ein Netzwerk zu verbessern.

P

Palette
Eine Tabelle in einer indizierten, farbigen 8-Bit-Datei, die die Farbinformationen für die Pixel in dem Bild bereitstellt.
Pfadname
Hinweise auf eine Datei mit Hilfe einer Nomenklatur, in der Verzeichnishierarchien und Dateinamen durch Slashes getrennt werden.
Pixel
Ein kleines Quadrat in einer Bitmap-Grafik.
PNG
Portable Network Graphic; ein Grafikdateiformat, das Unterstützung für 8-Bit- und 24-Bit-Bilder bietet.

Q

QuickTime
Ein Audio- und Video-Format, entwickelt von Apple Computer und verfügbar für alle Plattformen.

R

RGB-Farbe
Ein Farbsystem, das Farben auf der Grundlage von Kombinationen aus rotem, grünem und blauem Licht beschreibt.
Root-Verzeichnis
Das oberste Verzeichnis einer Website. Das Root-Verzeichnis (Wurzelverzeichnis) enthält alle Dateien und Unterverzeichnisse einer Site.

S

Schachteln
Ein HTML-Element in ein anderes platzieren. HTML und XHTML definieren Regeln, welche Elemente geschachtelt werden dürfen.
serifenlose Schrift
Eine Schrift mit geraden Zeichenabschlüssen ohne. Helvetica ist ein Beispiel.
Serifenschrift
Eine Schrift, die horizontale Abschlussstriche an den Enden der Zeichen besitzt. Times ist ein Beispiel.

T

Tweening
Eine Funktion in Animationsprogrammen, die automatisch Frames zwischen einem Anfangs- und einem End-Frame erzeugt.

U

Unix
Ein mehrbenutzerfähiges Multitasking-Betriebssystem, das von den Bell Laboratories entwickelt wurde.
URL
Universal Resource Locator; die Adresse eines Dokuments oder einer Ressource im World Wide Web

V

Vektorbild
Eine Grafik, die mathematische Gleichungen benutzt, um Formen und Füllungen zu definieren. Die Grösse von Vektorbildern kann ohne Qualitätsverlust geändert werden.
verlustbehaftet
Eine Art der Komprimierung, bei der Daten entfernt werden, um die Dateigrösse zu verringern. Das JPEG-Format verwendet ein verlustbehaftetes Komprimierungsschema.

W

W3C
Das World Wide Web Consortium; eine Vereinigung von Unternehmen und Organisationen, die Standards für die Entwicklung des Web entwickelt.
Webpalette
Eine Gruppe von 216 Farben, die sowohl auf Mac- als auch Windows-Rechnern mit 8-Bit-Farbmonitoren verfügbar ist.

X

XHTML
Der neue HTML-Standard, der entsprechend den Regeln von XML geschrieben wurde.
XML
eXtensible Markup Language; eine Metasprache zum Erzeugen eigener Auszeichnungssprachen.
